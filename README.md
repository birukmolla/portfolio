## üîë Key Features & Technical Implementation

### 1. Performance-Optimized Scroll Animations
* **The Challenge**: Standard scroll listeners can cause "jank" or lag on lower-end mobile devices.
* **The Solution**: Implemented the `IntersectionObserver` API in `script.js` to trigger reveal animations only when elements enter the viewport, significantly reducing CPU overhead.

### 2. Scalable Modular Styling
* **The Challenge**: Managing styles for a complex multi-section page can lead to bloated CSS files.
* **The Solution**: Utilized a `:root` variable system in `style.css` to manage a global design token system (colors, spacing, shadows), making the entire site easy to theme and maintain.

### 3. Integrated Project & Certification Showcases
* Features a dynamic modal system for enlarging certifications and detailed project cards that link to individual repositories, including the Advanced Document Management System and SafeScript.
